This applet demonstrates hypothesis testing. The purple distribution is the null distribution, and the hypothesized real population sampling distribution (the distribution you would get if you repeatedly sampled from the population and plotted the sample statistics) is also purple because in this case the population parameter equals the null parameter. In reality we don't know what the actual parameter value is and so we wouldn't know when this occurs. (A bootstrap distribution is an estimate of it, but the bootstrap distribution is centered around the sample statistic not the population parameter).

THE REAL POPULATION PARAMETER IS EQUAL TO THE NULL VALUE!

The darker purple represents the region of statistic values where the inference matches reality. The lighter purple represents the region of statistic values where a false negative error would be made. The dashed green line (or lines if you have checked both-tails) marks the statistic cutoff value that gives the specified significance level (or false alarm/type I error rate).

There are 2 possible outcomes when doing a hypothesis test - you either reject the null or do not reject it. Whether that is good or bad depends on whether the real parameter value actually equals the null value. In the scenario here, the real parameter value equals the null value:

GOOD: The real parameter value the null value and you decide that your data is consistent/agrees with it - you fail to reject it because your sample statistic falls to the left of the green line. This rate is known as the true negative rate. This is the fraction of the sampling distribution that is dark purple

FALSE ALARM: The real parameter value equals the null value but you decide that your data disagrees with it and you reject it because your sample statistic falls to the right of the green line. This is also called the Type I error rate or the false positive rate. This is the fraction of the sampling distribution that is light purple.

- Move the sliders around to see how the rates of error change depending on where you set your significance level cutoff, your sample size, and what the actual (but, in reality, unknown!) real parameter value is (the slider called "Real parameter value").
- If you press the triangles, the applet will animate that slider.
