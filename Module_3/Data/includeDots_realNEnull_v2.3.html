Instructions/Explanations (scroll down or expand the window for more)
This applet demonstrates hypothesis testing. The purple distribution is the null distribution, and the orange distribution is the hypothesized real population sampling distribution (the distribution you would get if you repeatedly sampled from the population and plotted the sample statistics). In reality we don't know what the actual parameter value is and so we don't know this sampling distribution. (A bootstrap distribution is an estimate of it, but the bootstrap distribution is centered around the sample statistic not the population parameter).

Currently, the real population parameter does not equal the null value

The darker orange represents the region of statistic values where the inference matches reality. The lighter orange represents the region of statistic values where a false negative error would be made. The dashed green line (or lines if you have checked both-tails) marks the statistic cutoff value that gives the specified significance level (or false alarm/type I error rate).

There are 2 possible outcomes when doing a hypothesis test - you either reject the null or do not reject it. Whether that is good or bad depends on whether the real parameter value actually equals the null value. In the scenario here, the real parameter value does not equal the null value:

GOOD: The real parameter value does not equal the null value and you decide that your data disagrees with with the null hypothesis and you reject it because your sample statistic falls to the right of the green line. This is also called the power of the hypothesis test (also the true positive rate). This is the fraction of the sampling distribution that is dark orange..

MISSED OPPORTUNITY: The real parameter value does not equal the null value but you decide that your data is consistent/agrees with the null hypothesis - you fail to reject it because your sample statistic falls to the left of the green line. This is also called the Type II error rate or the false negative rate. This is the fraction of the alternative distribution that is light orange.

- Move the sliders around to see how the rates of error change depending on where you set your significance level cutoff, your sample size, and what the actual (but, in reality, unknown!) real parameter value is (the slider called "Real parameter value").
- If you press the triangles, the applet will animate that slider.
